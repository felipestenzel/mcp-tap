# mcp-tap

**The last MCP server you install by hand.**

Discover, install, and configure MCP servers from inside your AI assistant.

## Install

```bash
uvx mcp-tap
```

## What can it do?

| You say | mcp-tap does |
|---------|-------------|
| "Find me an MCP for PostgreSQL" | Searches the MCP Registry, shows options |
| "Set up the official postgres server" | Installs, configures, validates connection |
| "What MCP servers do I have?" | Lists all configured servers |
| "Remove the slack MCP" | Removes from config cleanly |
| "Test my postgres connection" | Spawns server, connects, lists tools |
